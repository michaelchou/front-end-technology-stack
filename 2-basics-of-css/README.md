## 背景图像区域

### background-clip 属性

- background-clip 属性制定背景`绘制区域`
- 语法
  ```css
  background-clip: border-box; // 背景裁剪到边框盒
  background-clip: padding-box; // 背景裁剪到内边距
  background-clip: context-box; // 背景裁剪到内容框
  ```
- 浏览器支持
  - `IE9+、FireFox、Chrome、Safari、Opera`
